 2    |L(2)  |D(1)  |12    |3       |p = c
 3    |D(1)  |R(3)  |5     |4       |
 4    |R(3)  |D(1)  |      |4       |
 5    |D(1)  |R(3)  |3     |4       |
 6    |R(3)  |U(0)  |8     |3       |p = c
 7    |U(0)  |R(3)  |      |3       |p = c
 8    |R(3)  |U(0)  |6     |3       |p = c
 9    |U(0)  |L(2)  |11    |2       |
10    |L(2)  |U(0)  |      |2       |
11    |U(0)  |L(2)  |9     |2       |
12    |L(2)  |D(1)  |2     |3       |p = c
13    |L(2)  |R(3)  |      |5       |p = c
14    |D(1)  |U(0)  |      |1       |a =-3
15    |R(3)  |L(2)  |      |5       |p = c
16    |U(0)  |D(1)  |      |1       |a =-3

操作：

    1.所有纵向点 设为 -1； 横向点 设为 -2
    2.修改方向，需要重新修改标记

注：

	Prev：前一个点 Curr：当前点
	sum： 前后两点的数字和
	p=c： 将当前点的方向赋值给前一个点的方向
	a=-3: p,c 均设为-3标记 

总结：

	需要修改方向的条件：
	p + c = 3 或 p + c = 5
